# Intermediate Data Files

This folder (`data/intermediate/`) contains files generated by pipeline steps 1-3. These files are NOT tracked in Git due to size.

**Note:** This directory is a symbolic link to Google Drive for easy sharing and large file storage.

## Contents (~658 MB)

### Cleaned Data
- **h1b_fy21_24_cleaned.csv** (334 MB)
  - Output of step 3 (01_data_cleaning.R)
  - Cleaned H-1B data with merged LCA information
  - Includes SOC codes, wage levels, geocoding prep

### Checkpoint Files
- **step1_lca_processed.RData** (98 MB)
  - Output of step 1 (simplified pipeline)
  - Processed LCA data from 2015-2024

- **step2_foia_processed.RData** (77 MB)
  - Output of step 2 (simplified pipeline)
  - Processed FOIA H-1B petition data

### LCA Consolidated Files
- **lca_2020_2024.RData** (47 MB)
  - Recent quarterly LCA data

- **lca_2015_2024.RData** (101 MB)
  - Complete LCA data for all years

## Usage
These files are automatically generated when running the pipeline. You only need them if:
- Skipping early pipeline steps
- Debugging specific stages
- Collaborating with checkpoint recovery
