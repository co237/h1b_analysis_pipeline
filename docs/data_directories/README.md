# Data Directories Documentation

This folder contains documentation for the data directories used in the H-1B analysis pipeline.

## Overview

The pipeline uses three main data directories that are **symlinked to Google Drive** for easy sharing and large file storage:

- **`data/raw/`** → Original source data files (~7.1 GB)
- **`data/intermediate/`** → Generated intermediate files (~658 MB)
- **`data/processed/`** → Final processed datasets (~350 MB)

Because these are symbolic links, **Git cannot track files within them**. This documentation explains what files exist in each directory.

## Documentation Files

- **[RAW_DATA.md](./RAW_DATA.md)** - Source data files required to run the pipeline
- **[INTERMEDIATE_DATA.md](./INTERMEDIATE_DATA.md)** - Files generated by Scripts 01-03
- **[PROCESSED_DATA.md](./PROCESSED_DATA.md)** - Final datasets from Scripts 04-05

## Why Symbolic Links?

The data directories are symlinked to Google Drive to:
1. Share large files easily across team members
2. Keep the Git repository small and fast
3. Avoid accidental commits of large data files
4. Enable collaboration without data duplication

## Setting Up

To set up the data directories on a new machine:

1. Download the shared Google Drive folder
2. Create symbolic links in your local repository:
   ```bash
   ln -s /path/to/GoogleDrive/h1b_analysis_pipeline_data/raw data/raw
   ln -s /path/to/GoogleDrive/h1b_analysis_pipeline_data/intermediate data/intermediate
   ln -s /path/to/GoogleDrive/h1b_analysis_pipeline_data/processed data/processed
   ```

See the main [README.md](../../README.md) for more details.
