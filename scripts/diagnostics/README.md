# Diagnostic Scripts

This directory contains diagnostic and verification scripts used during development and debugging.

## Files

- **DIAGNOSTIC_missing_wages.R**: Analyzes patterns in missing wage data
- **DIAGNOSTIC_soc_oflc_match.R**: Checks SOC code matching between FOIA and OFLC data
- **VERIFY_soc_crosswalk_fix.R**: Verifies SOC crosswalk corrections

These scripts are not part of the main pipeline but can be useful for troubleshooting or verifying data quality.
